# CodeAlpha_Basic_Chatbot

A precise, interactive, and rule-based conversational AI assistant built entirely in Python for the CodeAlpha Internship program. 

This chatbot operates directly within the terminal interface, featuring real-time processing indicators, smooth typing text animations, and automated timezone handling for accurate local clock queries.

---

## ✨ Features

- **Interactive Conversations:** Smoothly handles standard greetings, identity questions, and general inquiries using clean tuple-key parsing logic.
- **Dynamic Local Time & Date:** Automatically synchronizes and fetches the precise local time and date for the `Asia/Kolkata` (IST) timezone, resolving typical cloud-environment time differences.
- **Randomized Joke Generator:** Houses a built-in pool of programming jokes, returning a completely unique single joke every time it is prompted.
- **Visual Terminal Effects:** Simulates a vintage typing delay effect and an animated `Bot is typing...` processing state for a clean user experience.
- **Graceful Termination:** Safely exits execution structures without terminal tracebacks when triggers like `bye`, `exit`, or `quit` are submitted.

---

## 🛠️ Commands Supported

Once running, you can type any of the following triggers to interact with the bot:
- **Greetings:** `hello`, `hi`, `hey`, `yo`
- **Identity / Info:** `who are you`, `what is your name`, `help`
- **Dynamic Data:** `time`, `clock`, `date`, `calendar`
- **Entertainment:** `joke`, `tell me a joke`, `make me laugh`
- **Exit Session:** `bye`, `goodbye`, `exit`, `quit`
